## Overview

This project automates downloading a dataset from Kaggle and uploading it to Google BigQuery.

## Modules

1. `1_kaggle_download.py` - Downloads the dataset from Kaggle.
2. `2_bigquery_upload.py` - Uploads the dataset to BigQuery.
3. `3_automate_etl.py` - Automates the ETL process.

Please refer to the [Setup Guide](setup.md) for installation instructions and the [Usage Instructions](usage.md) for detailed usage.
